django-admin --version ==> Version django 
django-admin startproject --help ==> Para saber que sigue 
python manage.py runserver ==> levanta el py django
python manage.py migrate ==> ejecutará todas las migraciones pendientes de bd
python manage.py showmigrations ==>  muestra las migraciones y las que no se ejecutaron en la bd no estaran seleccionadas


* NOTA *
Para cualquer comando se puede añadir al final el sub-comando --help para ver las opciones de ese comando.
